Talks
=====

To view the presentation
```sh
❯ cd talks/angular-ui-router
❯ grunt server
```

To run the example
```sh
❯ cd talks/angular-ui-router/example
❯ grunt server
```

Some examples are step-by-step. To see code for each step checkout the git branch.
```sh
❯ git checkout step-1
❯ git checkout step-2
```
